# TaskPriority

### 概要
緊急度と重要度から優先度がわかるタスク管理アプリ

### 開発環境/技術選定理由
* Kotlin：Javaよりも簡潔にコードを書けることから、継続的な運用に適していると考えたから。
* Realm：記録系アプリのためオフラインで使う場面が多いと考えたから。またSQLiteよりも高速なため選択。
* AndroidStudio：主要な開発環境で、Android開発に必要なツールがすべて統合されているかつテストがしやすいことから選んだ。

### 機能
* タスク一覧画面：タスク名と緊急度/重要度が一覧で確認できる画面。優先度が高いほどタスクの色が濃く表示されるので、優先度が高い順に並べると綺麗なグラデーションに見えるようになっています。
* タスク追加/編集画面：タスク名や完了期限、緊急度と重要度の追加/編集、タスク完了、タスク削除を行う画面です。
